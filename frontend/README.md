# Frontend for LiveCourierTrackingIP

This directory contains the React-based frontend for the LiveCourierTrackingIP project. It uses Vite for fast builds and Tailwind CSS for styling.

### Installation
#### 1. Navigate to `frontend` directory:
```bash
cd frontend
```
#### 2. Install dependencies:
```bash
npm install
```

#### 3. Start Dev Server:
```bash
npm run dev
```

### Build for Production
To build the production-ready files:
```bash
npm run build
```
The files will be output to the `dist/` folder.
